# 📦 StockFlow

StockFlow is a lightweight, web-based inventory management system built for small businesses that need speed, clarity, and control — without the complexity of enterprise ERP software.

It helps you track stock levels, low inventory, and expiring products in a clean, mobile-friendly interface.

## 🎯 Why StockFlow?

Many small businesses still manage inventory using spreadsheets or manual records, which leads to stockouts, expired products, and wasted time.

StockFlow is built to answer one critical question instantly:

What needs my attention today?

---

## ✨ Core Features (MVP)

Dashboard overview  
- Total products  
- Low stock  
- Expiring soon  
- Out of stock  

Needs Attention list  
- Highlights risky items first  
- Clear stock and expiry indicators  

Product management  
- Quantity tracking  
- Minimum stock levels  
- Optional expiry dates  

Mobile-first design  
- Built for daily operational use  

Barcode scanning and automation workflows are planned for future versions.

---

## 🧱 Tech Stack

Frontend: React + TypeScript  
Build Tool: Vite  
Styling: Tailwind CSS + shadcn-ui  
Backend: Supabase (PostgreSQL, Auth)  
Deployment: Lovable / Vercel  

---

## 🛠️ Local Development

Prerequisites  
- Node.js (v18+ recommended)  
- npm  

Setup steps  
1. git clone https://github.com/your-username/stockflow.git  
2. cd stockflow  
3. npm install  
4. npm run dev  

The app will run at  
http://localhost:5173

---

## 🔐 Environment Variables

Create a file named .env in the project root with the following keys:

VITE_SUPABASE_URL=your_supabase_project_url  
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key  

Never commit .env files to GitHub.
